# os-transport

Rust CLI tool for parsing OpenSearch's binary transport protocol (port 9300).

## Project Structure

```
src/
├── main.rs                 — CLI (clap): decode, read commands
├── protocol/               — Core protocol parsing
│   ├── vint.rs             — VInt/VLong/ZLong decoders
│   ├── status.rs           — Status byte flags
│   ├── version.rs          — Version ID (OS/ES detection, semver)
│   ├── string.rs           — Length-prefixed string decoding
│   ├── header.rs           — 23-byte fixed header
│   ├── var_header.rs       — Variable header (thread context, action name)
│   └── message.rs          — Full message envelope
└── capture/
    ├── pcap.rs             — Pcap file reader (uses pcap-parser + etherparse)
    └── reassembly.rs       — TCP stream reassembly
```

## Test Data

Captures live in `testdata/captures/`. Generated by running tcpdump against a local cluster:

```bash
./scripts/capture.sh
```

This captures traffic on ports 9300-9302 (transport ports for a 3-node local cluster).
